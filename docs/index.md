## Проблема

У тебя десятки задач из разных сфер жизни — работа, семья, финансы, здоровье, быт. Они все где-то записаны. Но каждое утро ты открываешь список и не знаешь с чего начать.

Стандартные таск-менеджеры хорошо **хранят** задачи. Но не помогают **принимать решения**.

Результат: фоновая тревога, прокрастинация, задачи делаются только когда становятся критически срочными.

---

## Решение

Ordo — не инструмент. Это советник.

Ты выгружаешь всё из головы и получаешь чёткий ответ каждый день: *вот что делать сегодня и почему*.

**Три роли Ordo:**

**Секретарь** — принимает всё что ты говоришь, записывает, структурирует. Быстро, без трения. Вспомнил задачу в душе — сказал в Telegram — забыл.

**Планировщик** — понимает контекст всех твоих задач, расставляет приоритеты сам. Без участия пользователя. Знает про дедлайны, ожидания, зависшие задачи.

**Диспетчер** — каждый день говорит конкретно: вот три задачи, делай их, вот почему именно они.

## Один день с Ordo

**07:30 — Просыпаешься**

```
Бот:  🌅 Доброе утро! Вот твой план на сегодня:

      1. Отправить договор клиенту — дедлайн сегодня
      2. Записаться к стоматологу — ждёшь уже 2 недели
      3. Позвонить маме — сам откладываешь

      Что убрать из плана сегодня?

      [Убрать договор]  [Убрать стоматолога]  [Убрать маму]  [Всё ок]
```

**09:00 — По дороге на работу**

```
Ты:   Надо встретиться с Алексеем на следующей неделе, обсудить бюджет

Бот:  📋 Встреча с Алексеем — обсудить бюджет
      📅 На следующей неделе
                                [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

**12:30 — Сделал задачу**

```
Ты:   Отправил договор

Бот:  ✅ Договор закрыт. Прогресс сегодня: 1 из 3.
      Следующее: Записаться к стоматологу.
```

**После обеда**

```
Ты:   Отправил резюме в компанию X, жду ответа до пятницы

Бот:  📋 Отправил резюме в компанию X
      ⏸ Статус: В ожидании
      💬 Причина: жду ответа
      📅 До: пятница
                                [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

**21:00 — Вечерний разбор**

```
Бот:  🌙 Вечерний разбор

      Зависшие задачи:
      ⏸ Позвонить в банк — ждёт 12 дней. Что делаем?
      [✅ Готово]  [📅 Перенести]  [❌ Удалить]

      ⏸ Дизайн сайта — ждёт ответа от Марины, срок вышел. Что делаем?
      [✅ Готово]  [📅 Перенести]  [❌ Удалить]

      Итог дня: 2 из 3 задач выполнено. Хорошо.
```

## Ключевые возможности

### Говоришь — бот понимает

```
Ты:   Нужно купить молоко, хлеб и яйца

Бот:  Нашёл 3 задачи. Создаём по одной:

      📋 Купить молоко
      [✅ Создать]  [⏭ Пропустить]
```

```
Ты:   Хочу сделать ремонт ванной

Бот:  📁 План: Ремонт ванной

      ├─ Составить смету
      │   └─ Обмерить помещение
      │   └─ Запросить цены у мастеров
      ├─ Выбрать плитку и сантехнику
      ├─ Найти бригаду
      └─ Закупить материалы
                                [✅ Создать план]  [❌ Отмена]
```

### /focus — что делать прямо сейчас

```
Ты:   /focus

Бот:  🎯 Сейчас делай:
      Отправить отчёт Дмитрию

      Почему: дедлайн сегодня, ты сам поставил высокий приоритет.
      Время: ~30 минут.

      [✅ Готово]  [⏭ Пропустить]
```

### Статус "Возможно"

Задачи которые не готов удалять, но и не обязуешься делать.

```
Ты:   Может когда-нибудь выучить испанский

Бот:  📋 Выучить испанский
      💭 Статус: Возможно

      Бот спросит раз в неделю: "Всё ещё думаешь об этом?"
      Переводишь в задачу или удаляешь — без вины.
```

### /progress — прогресс по планам

```
Ты:   /progress

Бот:  📊 Прогресс за неделю:

      Сделано: 11 задач ✅
      В работе: 4 задачи

      Планы:
      💒 Свадьба — 5/14 задач (36%)
      🔨 Ремонт — 3/10 задач (30%)
      💼 Инвестиции — 2/5 задач (40%)

      Что проседает: Ремонт — нет изменений 8 дней.
```
