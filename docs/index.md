## Узнаёшь себя?

Пятница, 21:00. Ты хочешь просто лечь и ни о чём не думать.

Но в голове прокручивается: *отчёт Дмитрию... записаться к врачу... написать подрядчику... подарок Асель... разобраться с ипотекой...*

Ты не можешь выключиться — потому что где-то там точно есть что-то срочное. Ты просто не помнишь что именно.

Задачи накапливаются. Делаешь только то что уже горит. Остальное висит фоном и создаёт постоянную тревогу. Даже когда отдыхаешь — не чувствуешь что можешь.

**Ты пробовал с этим что-то сделать:**

## Один день с Ordo

<div class="timeline">

<div class="timeline-step">
<div class="timeline-time">9:00 — Утренний план</div>

```
Ordo:  📅 План на Пт 25 апреля

       ✅ Запланировано: 3
       🤖 Рекомендует AI: 2

       [📋 Запланировано (3)]  [🤖 Рекомендации (2)]  [📅 Другой день]
```

Нажимаешь "Рекомендации" — видишь что AI выбрал и почему. Принимаешь или корректируешь.

*Не нужно думать с чего начать. Ordo уже решил.*
</div>

<div class="timeline-step">
<div class="timeline-time">10:00 — По дороге на работу</div>

```
Ты:    🎙 [голосовое] "Надо купить подарок Асель, у неё день рождения через неделю"

Ordo:  🎙 Распознано: "Надо купить подарок Асель, у неё день рождения через неделю"

       ✅ Купить подарок Асель — сохранено
       📅 2 мая

       [🤖 Предложить шаги]  [📋 Открыть задачу]
       [✏️ Изменить]  [🗑 Отменить]  [📅 Открыть план на 2 мая]
```

Мысль записана. Голова свободна. Не нужно останавливаться, открывать приложение, заполнять форму.
</div>

<div class="timeline-step">
<div class="timeline-time">В течение дня</div>

```
Ты:    Записался к стоматологу на вторник

Ordo:  ✅ Записался к стоматологу — сохранено
       ⏸ Статус: Ожидаю
       📅 Приём: вторник
```

Слово "записался" — бот сам поставил статус "Ожидаю". Не напомнит раньше времени.

```
Ты:    Что у меня запланировано на эту пятницу?

Ordo:  📅 Задачи на 25 апреля (3):
       1. Написать отчёт
       2. Позвонить в банк
       3. Встреча с командой
```

Спрашиваешь текстом или голосом — бот отвечает сразу.
</div>

<div class="timeline-step">
<div class="timeline-time">21:00 — Вечерний разбор</div>

```
Ordo:  🔍 Вечерний разбор — 8 задач

       ⏰ Просрочено — 1
       ⏳ В ожидании — 2
       📋 Без даты — 5

       [⏰ Просрочено (1)]  [⏳ В ожидании (2)]
       [📋 Без даты (5)]    [▶️ Разобрать всё]
```

Нажимаешь "Разобрать всё" — бот показывает по одной. Назначаешь дату, откладываешь или удаляешь.

*Три нажатия — голова чистая перед сном.*
