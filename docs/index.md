## Что отличает Ordo

**Говоришь — он понимает контекст**

Не нужно думать в каком статусе создать задачу. Ordo сам разбирает что ты имеешь в виду:

```
Ты:   Записался к стоматологу на пятницу

Бот:  📋 Записался к стоматологу
      ⏸ Статус: Жду (не напомню раньше пятницы)
      📅 До: пятница

      [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

Фраза "записался" — и бот сам поставил статус "Жду". Не будет трогать задачу до нужного момента.

---

**Большая цель → конкретный план за секунду**

```
Ты:   Хочу сделать ремонт ванной

Бот:  📁 План: Ремонт ванной

      ├─ Составить смету
      │   └─ Обмерить помещение
      │   └─ Запросить цены у мастеров
      ├─ Выбрать плитку и сантехнику
      ├─ Найти бригаду
      └─ Закупить материалы

      [✅ Создать план]  [❌ Отмена]
```

---

**Утренний план — с объяснением почему**

Не просто список. Ordo объясняет логику — чтобы ты мог согласиться или поменять:

```
Бот:  🌅 Доброе утро! На сегодня:

      1. 💍 Позвонить маме насчёт свадьбы
         → свадьба через 2 месяца, блокирует выбор даты

      2. 🏠 Написать подрядчику
         → ты ждал его ответа — он ответил вчера

      3. ⏸ Проверить доставку с WB
         → срок ожидания вышел

      [✅ Берусь]  [✏️ Изменить список]
```
