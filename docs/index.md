## Не инструмент. Советник.

Большинство таск-менеджеров хорошо хранят задачи.
Ordo помогает понять **что делать прямо сейчас**.

**🗂 Секретарь**

Говоришь или пишешь — он записывает. Сам разбирает дату, статус, категорию. Никаких обязательных полей.

**🧠 Планировщик**

Анализирует все твои задачи и понимает что важнее — на основе дедлайнов, контекста и зависимостей. Без твоего участия.

**📋 Диспетчер**

Каждое утро присылает 3 задачи на день и объясняет почему именно они. Ты убираешь что не хочешь — и идёшь делать.

---

## Знакомо?

В голове постоянно висят десятки задач — работа, семья, деньги, здоровье, бытовые дела. Ты никогда не чувствуешь что сделал достаточно.

Задачи копятся. Откладываешь пока не станет срочно.
Делаешь на энергии тревоги. И снова ждёшь следующего дедлайна.

**Проблема не в лени. Проблема в том что нет системы которой ты доверяешь.**

Ordo забирает этот хаос из головы. Ты выгружаешь всё — он следит за тем чтобы ничего не провисало.

## Один день с Ordo

**7:30 — Просыпаешься**

```
Бот:  🌅 Доброе утро!

      На сегодня предлагаю:

      1. 💍 Позвонить маме насчёт свадьбы
         → свадьба через 2 месяца, блокирует выбор даты

      2. 🏠 Написать подрядчику
         → ты ждал его ответа — он ответил вчера

      3. ⏸ Проверить доставку с WB
         → срок ожидания вышел

      [✅ Берусь]  [✏️ Изменить список]
```

Убираешь что не хочешь сегодня — голосом или кнопкой.

---

**10:00 — По дороге на работу**

```
Ты:   Надо купить подарок Асель, у неё день рождения через неделю

Бот:  📋 Купить подарок Асель
      📅 До: 3 апреля
      Добавить в сегодняшний список?
      [Да]  [Нет, потом]
```

Мысль записана. Голова свободна.

---

**12:30 — Обед**

```
Ты:   Сделал — позвонил маме

Бот:  ✅ Отлично!
      Сегодня: 1 из 3
      💍 Свадьба: 5 из 14 задач готово
```

---

**21:00 — Вечер**

```
Бот:  🔍 Вечерний разбор

      ⏸ Жду ответа от HR (6 дней, без даты)
         Пора напомнить им?

      📋 Разобраться с инвестициями (21 день)
         [Разбить на шаги]  [В maybe]  [Удалить]
```

Три нажатия — голова чистая перед сном.

## Примеры диалогов

### Говоришь — бот понимает

```
Ты:   Нужно купить молоко, хлеб и яйца

Бот:  📋 Купить продукты
      ☐ Молоко
      ☐ Хлеб
      ☐ Яйца

      [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

```
Ты:   Хочу сделать ремонт ванной

Бот:  📁 План: Ремонт ванной

      ├─ Составить смету
      │   └─ Обмерить помещение
      │   └─ Запросить цены у мастеров
      ├─ Выбрать плитку и сантехнику
      ├─ Найти бригаду
      └─ Закупить материалы
                                [✅ Создать план]  [❌ Отмена]
```

### Автоопределение ожидания

```
Ты:   Отправил резюме в компанию X, жду ответа до пятницы

Бот:  📋 Отправил резюме в компанию X
      ⏸ Статус: В ожидании
      💬 Причина: жду ответа
      📅 До: пятница
                                [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

### /focus — что делать прямо сейчас

```
Ты:   /focus

Бот:  🎯 Сейчас делай:
      Отправить отчёт Дмитрию

      Почему: дедлайн сегодня, ты сам поставил высокий приоритет.
      Время: ~30 минут.

      [✅ Готово]  [⏭ Пропустить]
```

### Статус "Возможно"

```
Ты:   Может когда-нибудь выучить испанский

Бот:  📋 Выучить испанский
      💭 Статус: Возможно

      Бот спросит раз в неделю: "Всё ещё думаешь об этом?"
      Переводишь в задачу или удаляешь — без вины.
```

### /progress — прогресс по планам

```
Ты:   /progress

Бот:  📊 Прогресс за неделю:

      Сделано: 11 задач ✅
      В работе: 4 задачи

      Планы:
      💒 Свадьба — 5/14 задач (36%)
      🔨 Ремонт — 3/10 задач (30%)
      💼 Инвестиции — 2/5 задач (40%)

      Что проседает: Ремонт — нет изменений 8 дней.
```
