# Запись задач

Ordo записывает задачи так же просто как ты говоришь или пишешь — без форм, без обязательных полей, без лишних шагов.

---

## Голосовые сообщения

Запись голосовым — самый быстрый способ. Отправляешь голосовое — Ordo расшифровывает речь и сразу разбирает задачу.

```
Ты:    🎙 [голосовое] "Купить подарок Асель, у неё день рождения через неделю"

Ordo:  🎙 Распознано: "Купить подарок Асель, у неё день рождения через неделю"

       ✅ Купить подарок Асель — сохранено
       📅 2 мая

       [🤖 Предложить шаги]  [📋 Открыть задачу]
       [✏️ Изменить]  [🗑 Отменить]  [📅 Открыть план на 2 мая]
```

Задача сохранена немедленно. Можно исправить или отменить — кнопки прямо под сообщением.

## Что бот определяет сам

Не нужно указывать категорию, статус или формат даты — бот разберёт из контекста.

| Что ты сказал | Что бот определил |
|---|---|
| "...на пятницу" | 📅 Дата: ближайшая пятница |
| "...до 15 апреля" | 📅 Дата: 15 апреля |
| "...через неделю" | 📅 Дата: через 7 дней |
| "Купить молоко" | 📁 Категория: Покупки |
| "Написать отчёт" | 📁 Категория: Работа |
| "Записаться к врачу" | 📁 Категория: Здоровье |
| "Заплатить налоги" | 📁 Категория: Финансы |

## Статус "Ожидаю" — автоматически

Если ты уже совершил действие и ждёшь результата — бот сам ставит статус **Ожидаю**. Не будет напоминать лишний раз.

```
Ты:    Записался к стоматологу на пятницу

Ordo:  ✅ Записался к стоматологу — сохранено
       ⏸ Статус: Ожидаю
       📅 Приём: пятница
```

```
Ты:    Заказал вешалки на WB, придут 25 апреля

Ordo:  ✅ Заказал вешалки на WB — сохранено
       ⏸ Статус: Ожидаю
       💬 Жду: доставка заказа с WB
       📅 До: 25 апреля
```

```
Ты:    Отправил резюме в компанию, жду ответа

Ordo:  ✅ Отправил резюме — сохранено
       ⏸ Статус: Ожидаю
       💬 Жду: ответ от компании
```

**Признаки которые бот распознаёт:** "записался", "заказал", "отправил", "договорился", "жду", "должны позвонить", "машина в сервисе", "посылка в пути"...

## Несколько задач сразу

Можно перечислить через запятую — бот создаст каждую отдельно.

```
Ты:    Задачи на неделю: позвонить в банк, продлить страховку, купить продукты

Ordo:  ✅ Позвонить в банк — сохранено
       ✅ Продлить страховку — сохранено
       ✅ Купить продукты — сохранено
```

---

## Повторяющиеся напоминания

Для регулярных задач — тренировки, таблетки, платежи.

```
Ты:    Каждый понедельник и среду тренировка в 19:00, напомни за 30 минут

Ordo:  ✅ Тренировка — повторяющееся
       🔄 Пн и Ср в 18:30 (напомню за 30 мин)
```

```
Ты:    Каждый день в 8 утра пить витамины

Ordo:  ✅ Пить витамины — повторяющееся
       🔄 Каждый день в 08:00
```

```
Ты:    1-го числа каждого месяца оплата аренды

Ordo:  ✅ Оплата аренды — повторяющееся
       🔄 1-е число каждого месяца в 09:00
```

## Управление голосом и текстом

Задачи можно не только создавать, но и управлять ими — без команд.

```
Удали задачу молоко
Отметь задачу отчёт как выполненную
Поставь задачу "позвонить в банк" на завтра
Покажи задачи на пятницу
Задачи без даты
Задачи по категории Работа
```
