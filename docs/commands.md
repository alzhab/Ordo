# Все команды и фразы

Ordo понимает естественный язык — большинство действий можно сделать голосом или текстом, без команд. Но команды тоже работают.

---

## Основные команды

| Команда | Что делает |
|---------|------------|
| `/start` | Начало работы / приветствие |
| `/add` | Добавить задачу текстом: `/add Купить молоко` |
| `/tasks` | Список всех задач с фильтрами |
| `/plan` | План на сегодня + AI рекомендации |
| `/review` | Разбор зависших задач |
| `/goals` | Список целей и проектов |
| `/settings` | Настройки: время уведомлений, категории |
| `/help` | Справка |

---

## Создание задач — что можно сказать

```
Купить молоко
Позвонить маме в воскресенье
Написать отчёт Дмитрию до 30 апреля
Записаться к окулисту, срок пятница
Оплатить счёт за квартиру сегодня
Разобраться с инвестициями
```

**Ожидание** (бот сам распознаёт):
```
Записался к стоматологу на вторник
Заказал вешалки на WB, придут 25 апреля
Отправил резюме в компанию X
Машина в сервисе
```

**Несколько задач сразу:**
```
Задачи на неделю: позвонить в банк, продлить страховку, купить продукты
```

**Повторяющиеся:**
```
Каждый день в 8 утра пить витамины
Каждый понедельник и среду тренировка в 19:00
1-го числа каждого месяца оплата аренды
Рабочие дни в 9:30 стендап
```

## Управление задачами голосом

```
Удали задачу молоко
Отметь задачу отчёт как выполненную
Поставь задачу "позвонить в банк" на завтра
Запланируй задачу ремонт на 15 мая
Переведи задачу X в ожидание, жду доставки
Напомни про задачу встреча завтра в 10:00
```

**Групповые операции:**
```
Все задачи выполнены
Первые 3 задачи — выполнить
Все задачи по категории Покупки — удалить
```

## Открыть план или разбор голосом

```
Покажи план на сегодня
План на завтра
Что у меня на пятницу?
Свободен ли я в среду?
План на 15 мая

Давай сделаем разбор
Покажи зависшие задачи
Открой разбор
```

**Убрать задачи из плана:**
```
Убери все задачи из плана на сегодня
Освободи 30 апреля
Очисти план на пятницу
```

```
Покажи цели
Покажи задачи цели Свадьба
Архивируй цель Ремонт
```

## Настройки голосом

```
Поставь план на 8 утра
Присылай разбор в 22:00
Не беспокой до завтра
Не беспокой до воскресенья
Выключи утренний план
Включи напоминания
```

## Категории голосом

```
Покажи задачи категории Работа
```

## Фильтры в /tasks

Открыв `/tasks` — видишь кнопки фильтрации:

- **Все** — все активные задачи
- **Ожидаю** — задачи в статусе waiting
- **🔄 Повторяющиеся** — регулярные напоминания
- **По целям** — задачи конкретной цели
- **По категории** — задачи по теме
- **🔍 Поиск** — поиск по названию

---

## Настройки (/settings)

- **Уведомления** — время утреннего плана и вечернего разбора, тихий режим
- **Категории** — создать, переименовать, удалить
- **О боте** — версия и информация
