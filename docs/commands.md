# Команды и примеры

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и краткая справка |
| `/help` | Инструкция по использованию |
| `/add [текст]` | Создать задачу напрямую |
| `/tasks` | Список задач с фильтрами |
| `/today` | Задачи на сегодня |
| `/plans` | Планы с прогрессом выполнения |
| `/settings` | Настройки и интеграции |

---

## Создание задачи

Просто напиши или скажи задачу — бот сам разберёт поля:

```
Купить молоко
Купить молоко, срок завтра
Записаться к врачу, срок пятница, приоритет высокий
Починить велосипед, категория хобби, приоритет низкий
Отправить отчёт, срок 25 марта, категория работа
```

Бот покажет превью с распознанными полями. Жми **✅ Создать** для подтверждения.

### Автоматический статус "В ожидании"

Если фраза подразумевает уже совершённое действие, бот автоматически ставит статус "В ожидании":

```
Записался к врачу на пятницу — жду приёма
Заказал запчасть на Wildberries
Отправил резюме в компанию X
```

## Голосовые команды

### Управление задачами

```
Удали задачу молоко
Переведи задачу ремонт в работу
Отметь задачу велосипед как выполненную
Поставь высокий приоритет задаче отчёт
Покажи задачи по категории работа
```

### Групповые операции

```
Все задачи готовы
Первые 3 задачи — в работу
Последние 5 задач — низкий приоритет
Половина задач выполнена
```

### Планы

```
Создай план "Ремонт кухни"
Хочу сделать ремонт кухни   →   AI создаёт план с задачами и шагами
Архивируй план ремонт
Покажи задачи плана работа
```

### Несколько задач сразу

```
Нужно купить молоко, хлеб и масло
Задачи на неделю: позвонить маме, заплатить за интернет, сдать отчёт
```

Бот создаёт задачи по одной — подтверждаешь или пропускаешь каждую.

## Фильтры в /tasks

| Фильтр | Описание |
|--------|----------|
| ▶️ В работе | Задачи со статусом "в работе" (по умолчанию) |
| ⏸ В ожидании | Задачи, где ждёшь результата |
| 🔲 Не начато | Новые задачи |
| ✅ Выполненные | Закрытые задачи |
| 📋 Все активные | Всё, кроме выполненных |
| 📁 По плану | Задачи конкретного плана |

Фильтры запоминаются между вызовами.

---

## Редактирование задачи

Открой задачу из списка и нажми ✏️ — можно редактировать название, описание, дату, приоритет, категорию, причину и срок ожидания.

---

## Подзадачи

В карточке задачи → кнопка **Шаги**:

- ➕ Добавить шаг вручную
- 🤖 Сгенерировать шаги через AI
- Нажать на шаг — отметить выполненным

---

## Категории

```
Создай категорию Здоровье
Удали категорию спорт
Покажи все категории
```

Категории создаются автоматически при первом упоминании или через `/settings`.
