# Как пользоваться

## Первый шаг

Не нужно учить команды. Просто напиши или скажи что хочешь сделать — как обычно говоришь.

```
Купить молоко до вторника
Записаться к врачу, срок пятница
Отправить отчёт Дмитрию, важно
Позвонить маме на выходных
```

Бот сам разберёт: задачу, дату, статус, категорию. Покажет превью — подтвердишь или поправишь.

---

## Голосовые сообщения

Работают так же как текст. Говоришь — бот сначала показывает транскрипцию, потом разбирает задачу.

```
[голосовое: "Надо купить подарок Маше на день рождения, у неё 15 апреля"]

Бот:  🎙 Транскрипция: "Надо купить подарок Маше на день рождения, у неё 15 апреля"

      📋 Купить подарок Маше
      📅 До: 15 апреля
                          [✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```

## Создание задач

### Одна задача

```
Сдать налоговую декларацию до 30 апреля

→ 📋 Сдать налоговую декларацию
  📅 30 апреля
```

### Несколько задач сразу

```
Задачи на неделю: позвонить в банк, продлить страховку, записаться к окулисту

→ Бот создаёт по одной — подтверждаешь или пропускаешь каждую:

  📋 Позвонить в банк
  [✅ Создать]  [⏭ Пропустить]
```

### Автоматический статус "В ожидании"

Когда действие уже совершено и ждёшь результата — бот определяет это сам:

```
Записался к стоматологу на пятницу

→ 📋 Записался к стоматологу
  ⏸ Статус: В ожидании
  📅 До: пятница

Заказал запчасть на Wildberries

→ 📋 Заказал запчасть
  ⏸ Статус: В ожидании
  💬 Причина: ждём доставку

Отправил резюме в компанию X, жду ответа до пятницы

→ 📋 Отправил резюме в компанию X
  ⏸ Статус: В ожидании
  💬 Причина: жду ответа
  📅 До: пятница
```

## Управление задачами голосом

```
Удали задачу молоко
Отметь задачу отчёт как выполненную
Переведи задачу ремонт в работу
Покажи задачи по категории работа
Покажи задачи на эту неделю
```

### Групповые операции

```
Все задачи готовы
Первые 3 задачи — закрыть
Половина задач выполнена
Все задачи из категории покупки — выполнены
```

## Статусы задач

| Статус | Смысл |
|--------|-------|
| 📋 **Todo** | Надо сделать, взял обязательство |
| ⏸ **Waiting** | Жду чего-то внешнего — ответа, доставки, встречи |
| 💭 **Maybe** | Возможно когда-нибудь, пока не обязуюсь |
| ✅ **Done** | Готово |

**Waiting — два типа:**
- Ждёшь дату → бот молчит до наступления срока
- Ждёшь событие без даты → бот поднимает в вечернем разборе через несколько дней

**Maybe** — задачи которые не хочешь удалять но и не готов делать. Раз в неделю бот спрашивает: *"Всё ещё думаешь об этом?"* — переводишь в задачу или удаляешь без вины.

## Планы

Plan — это большая цель, разбитая на задачи. Свадьба, ремонт, переезд, запуск продукта.

```
Создай план "Переезд в новую квартиру"

→ 📁 Переезд в новую квартиру
  Пока пустой. Добавляй задачи.

Хочу организовать свадьбу на 50 человек

→ 📁 Свадьба
  ├─ Выбрать дату и место
  │   └─ Составить список площадок
  │   └─ Уточнить бюджет
  ├─ Список гостей
  ├─ Найти ведущего
  ├─ Выбрать меню
  └─ Фотограф и видеооператор
                      [✅ Создать план]  [❌ Отмена]
```

Прогресс по планам — в `/progress`.

---

## Ассистент

### /morning — утренний план

Приходит автоматически в 9:00. Или вызови сам когда нужно.

```
/morning

Бот:  🌅 Доброе утро! Вот три задачи на сегодня:

      1. Ответить на письмо Кириллу — ждёт 4 дня
      2. Купить продукты — запланировано на сегодня
      3. Позвонить в страховую — срок вышел вчера

      Убрать что-то из плана?
      [Убрать задачу 1]  [Убрать задачу 2]  [Убрать задачу 3]  [Всё ок]
```

### /focus — что делать прямо сейчас

Когда не знаешь с чего начать.

```
/focus

Бот:  🎯 Прямо сейчас: Ответить на письмо Кириллу

      Почему: ждёт 4 дня, ты сам обещал ответить сегодня.

      [✅ Сделано]  [⏭ Пропустить]
```

### /review — вечерний разбор

Приходит автоматически в 21:00. Поднимает зависшие задачи.

```
/review

Бот:  🌙 Вечерний разбор

      Срок ожидания вышел:
      ⏸ Дизайн сайта — ждёшь ответа от Марины, срок вышел.
      [✅ Готово]  [📅 Перенести]  [❌ Удалить]

      ⏸ Посылка с Wildberries — заказал 10 дней назад.
      [✅ Получил]  [📅 Перенести]  [❌ Отмена]

      Итог дня: 3 задачи выполнено ✅
```

### /progress — итоги

```
/progress

Бот:  📊 Эта неделя:

      Выполнено: 11 задач ✅
      В ожидании: 3 задачи

      Планы:
      💒 Свадьба — 5 из 14 задач (36%)
      🔨 Ремонт — 3 из 10 задач (30%)

      Внимание: Ремонт — нет изменений 8 дней.
```

## Команды

| Команда | Когда использовать |
|---------|-------------------|
| `/tasks` | Полный список с фильтрами |
| `/today` | Задачи на сегодня |
| `/plans` | Все планы с прогрессом |
| `/morning` | Утренний план прямо сейчас |
| `/focus` | Что делать прямо сейчас |
| `/review` | Вечерний разбор прямо сейчас |
| `/progress` | Итоги недели |
| `/settings` | Настройки, Notion, категории |
| `/help` | Справка |

---

## Фильтры в /tasks

| Фильтр | Описание |
|--------|----------|
| 📋 Todo | Задачи которые надо сделать |
| ⏸ В ожидании | Ждёшь результата |
| 💭 Возможно | Maybe-задачи |
| ✅ Выполненные | История |
| 📁 По плану | Задачи конкретного плана |
| 🗂 По категории | Работа, здоровье, быт и т.д. |

Фильтр запоминается между вызовами.

## Подзадачи

В карточке задачи → кнопка **Шаги**:

```
Бот:  📋 Сделать презентацию

      Шаги:
      ☐ Собрать данные
      ☐ Написать структуру
      ☐ Оформить слайды

      [➕ Добавить шаг]  [🤖 AI предложит шаги]
```

Нажми на шаг — отметишь выполненным.

---

## Категории

```
Создай категорию Здоровье
Покажи задачи категории Работа
Удали категорию Хобби
```

Категория создаётся автоматически при первом упоминании — или через `/settings`.
