# Цели и проекты

Цель — это крупный проект который состоит из многих задач. Свадьба, ремонт, переезд, карьерный рост, запуск бизнеса.

Ordo помогает разбить большую цель на конкретные шаги — и держит всё в одном месте.

---

## Создать цель

### Пустая цель

Если ты уже знаешь что нужно сделать:

```
создай цель Ремонт в квартире
новый план: Обучение
```

```
Ты:    Создай цель "Ремонт в квартире"

Ordo:  ✅ Цель Ремонт в квартире создана!
       Добавляй задачи — голосом или через /tasks
```

### AI разбивает цель на задачи

Если не знаешь с чего начать — скажи Ordo что хочешь сделать. AI сам предложит план:

```
Ты:    Хочу организовать свадьбу на 50 человек в сентябре

Ordo:  📁 Свадьба — предлагаю план:

       1. Выбрать дату и площадку
          · Составить список площадок
          · Уточнить бюджет
          · Забронировать

       2. Список гостей и приглашения
          · Составить полный список
          · Отправить приглашения за 2 мес

       3. Организация дня
          · Найти ведущего
          · Выбрать меню с рестораном
          · Фотограф и видеооператор

       4. Детали
          · Наряды и образы
          · Цветы и декор
          · Транспорт для гостей

       [✅ Создать всё]  [❌ Отмена]
```

Нажимаешь "Создать всё" — Ordo создаёт цель и все задачи сразу.

---

## Задачи внутри цели

Любую задачу можно привязать к цели:

```
создай задачу "Выбрать меню" в цели Свадьба
добавь к плану Ремонт: купить краску
```

Или при создании задачи просто упомяни цель:

```
для свадьбы нужно найти ведущего
```

---

## Подзадачи (шаги)

В каждой задаче можно создать список шагов — что конкретно нужно сделать.

```
Ordo:  📋 Выбрать площадку для свадьбы
       Цель: Свадьба

       Шаги:
       ☐ Составить список площадок (минимум 5)
       ☐ Объехать топ-3 варианта
       ☐ Уточнить стоимость и условия
       ☐ Забронировать выбранную

       [☑ Отметить шаг]  [➕ Добавить шаг]  [🤖 AI предложит шаги]
```

Если шагов нет — нажми "AI предложит шаги" и Ordo сам добавит конкретный список.

---

## Просмотр целей

Команда `/goals` — список всех активных целей.

```
Ordo:  🎯 Цели (3)

       💒 Свадьба — 5 из 14 задач
       🏠 Ремонт в квартире — 2 из 8 задач
       📈 Финансы — 1 из 5 задач
```

Нажимаешь на цель — видишь все задачи по ней.

---

## Управление голосом

```
покажи цели
покажи задачи цели Свадьба
архивируй цель Ремонт
удали план Учёба
```

---

## Архивирование

Когда цель достигнута — архивируй. Задачи остаются в истории, цель исчезает из активного списка.

```
архивируй цель Свадьба
```

<div class="tip-box">
💡 <b>Как использовать цели максимально:</b> Скажи Ordo о большом плане который у тебя есть — "хочу переехать в новую квартиру" или "планирую запустить свой бизнес". AI разберёт его на конкретные первые шаги — и ты начнёшь двигаться, а не прокрастинировать.
</div>
