# Вечерний разбор

Каждый вечер Ordo сам поднимает задачи которые давно не двигались. Не нужно помнить о них — бот найдёт их сам и предложит разобраться.

---

## Как это выглядит

В настроенное время (по умолчанию 21:00) бот присылает сводку по зависшим задачам:

```
Ordo:  🔍 Вечерний разбор — 8 задач

       ⏰ Просрочено — 1
       ⏳ В ожидании — 2
       📋 Без даты — 5

       [⏰ Просрочено (1)]  [⏳ В ожидании (2)]
       [📋 Без даты (5)]    [▶️ Разобрать всё]
```

Нажимаешь на группу — видишь список. Нажимаешь "Разобрать всё" — бот показывает задачи по одной.

---

## Четыре группы задач

### ⏰ Просрочено

Задачи у которых прошла запланированная дата или истёк срок ожидания.

*Например: "Написать отчёт" был на прошлую пятницу — не сделал.*

### ⏳ В ожидании

Задачи со статусом "Ожидаю" у которых нет конкретной даты или дата уже прошла.

*Например: "Отправил резюме — жду ответа" — прошло уже 5+ дней.*

### 📋 Без даты

Обычные задачи которые ты создал, но не поставил на конкретный день и не трогал 7+ дней.

*Например: "Разобраться с инвестициями" — лежит в списке уже 2 недели.*

### 💤 Отложено

Задачи которые ты ранее пометил как "Отложить" — они не появляются в разборе 7 дней, а потом всплывают снова.

---

## Разбор задач по одной

Нажав "Разобрать всё" — листаешь задачи в слайдере:

```
Ordo:  📋 Разобраться с инвестициями
       Без даты — уже 21 день

       2 из 8

       [🗓 Сегодня]  [📅 Завтра]  [📆 На неделе]  [📅 Выбрать дату]
       [⏭ Отложить]  [🗑 Удалить]  [🔗 Открыть]

       [◀️]  [📋 К обзору]  [▶️]
```

**Действия:**

| Кнопка | Что происходит |
|---|---|
| 🗓 Сегодня / Завтра | Ставит задачу на конкретный день |
| 📆 На неделе | Ставит на ближайшую пятницу |
| 📅 Выбрать дату | Открывает календарь |
| ⏭ Отложить | Скрывает на 7 дней |
| 🗑 Удалить | Удаляет задачу |
| 🔗 Открыть | Открывает полную карточку задачи |

---

## Задачи в ожидании

Если у задачи истёк срок ожидания, бот показывает её в разборе:

```
Ordo:  ⏳ Отправил резюме в компанию X
       Жду: ответ от компании
       Срок ожидания вышел — прошло уже 7 дней

       1 из 3

       [🗓 Сегодня]  [📅 Завтра]  [📆 На неделе]  [📅 Выбрать дату]
       [⏭ Отложить]  [🗑 Удалить]  [🔗 Открыть]
```

Если ответ уже получен — ставишь дату "Сегодня" и звонишь/пишешь. Если ещё ждёшь — жмёшь "Отложить".

---

## Открыть разбор в любое время

Не обязательно ждать 21:00. Скажи или напиши:

```
давай сделаем разбор
открой разбор
покажи зависшие задачи
что у меня зависло
```

Или нажми `/review`.

---

## Если всё под контролем

Если нет задач требующих внимания — бот скажет об этом:

```
Ordo:  🎉 Всё под контролем!

       Нет задач требующих внимания.
       Все активные задачи уже запланированы.
```

---

## Настройка времени

По умолчанию разбор приходит в 21:00. Можно изменить:

```
присылай разбор в 22:00
выключи вечерний разбор
```

Или через `/settings` → Уведомления.

<div class="tip-box">
💡 <b>Совет:</b> Разбор — это не наказание за несделанные задачи. Это инструмент разгрузки головы. Даже если просто нажать "Отложить" на всё — это нормально. Главное — голова чистая перед сном.
</div>
